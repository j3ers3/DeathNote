### Function
- windows 基于白名单的反弹shell及后渗透攻击

- linux 反弹shell

- powershell 攻击模块

- python 反弹shell

- msfvenom 查询


### Install
```
兼容py2和py3

git clone https://github.com/j3ers3/DeathNote

cd DeathNote

pip install -r requirements.txt

python death.py
```
